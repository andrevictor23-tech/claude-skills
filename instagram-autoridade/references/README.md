# Pasta de Referências - instagram-autoridade

Esta pasta contém os recursos de identidade visual e histórico da skill.

## Arquivos Esperados

| Arquivo                  | Tipo       | Descrição                                              | Status     |
|--------------------------|------------|--------------------------------------------------------|------------|
| `paleta-cores.md`        | Markdown   | Paleta de cores e tipografia institucional             | ✅ Presente |
| `perfil-avatar.png`      | Imagem     | Foto/avatar do perfil Instagram                        | ⬜ Pendente |
| `assinatura.png`         | Imagem     | Marca d'água ou assinatura para rodapé dos materiais   | ⬜ Pendente |
| `relatorio-anterior.md`  | Markdown   | Resumo do relatório anterior (gerado automaticamente)  | ⬜ Pendente |

## Como Adicionar Arquivos

1. **Avatar:** Faça upload da foto de perfil do Instagram e salve como `perfil-avatar.png`
2. **Assinatura:** Faça upload de logo ou assinatura e salve como `assinatura.png`
3. **Relatório anterior:** Gerado automaticamente pela skill ao final de cada análise (confirmação do usuário)

## Notas

- A skill usa automaticamente o que encontrar nesta pasta
- Se um arquivo não existir, a skill usa os padrões definidos no `paleta-cores.md`
- O arquivo `relatorio-anterior.md` permite comparações entre períodos
